## Setup

Configure `.env` based on `.env.example`

Install npm packages
```bash
npm install
```

## Development

Start for development server
```bash
npm run dev
```

###  Lint

Check lint errors

```bash
npm run lint
```

## Build

Build into `dist`
```bash
npm run build
```
